# setting up a svelte-kit project

type the following into your terminal to create a new application
```cmd
npm init svelte@next appName
cd appName
npm i
```

optional : if you want to add tailwind:
```cmd
npx svelte-add tailwindcss
npm i
```

to build the source
```
npm run build
```

to run the source
```
npm run dev
```
